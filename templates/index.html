What is CNN ?

What is CNN ?

A Convolutional Neural Network is a Deep Learning algorithm which can take in an input image, assign importance to various objects in the image and be able to differentiate one from the other.

The pre-processing required in a CNN is much lower as compared to other classification algorithms. Each input image will pass it through a series of convolution layers. At the end model will predict the image.

Why I used CNN ?

There are some reasons that why I used CNN algorithm not other algorithm.

  • CNN has very high accuracy in image recognition problems
  • CNN is more powerful than anyother algorithm like ANN, RNN.
  • CNN required less pre-processing compared to other classification algorithms.
  • CNN can automatically detects the important features wisthout any human supervision.

Implementation

Step 1

Downloading the dataset : This is animal image classifier and I have downloaded the Animal - 10 image dataset for my project and I have downloaded it from kaggle. You can find dataset for your project also.

Step 2

Importing the libraries : We have to install necessary libraries befort importing them. In this project most important library is Tensorflow.

Step 3

Pre-processing : Before using the dataset we have to pre-process it. Pre-processing is data mining technique in which raw data is transform into the useful format.

Step 4

Normalization : Here our Animal - 10 image dataset is normalize on the scale of 0 to 1. Data normalization is part of pre-processing, in which data is minimized.

Step 5

Training dataset & Validation dataset : In this step our image dataset is splitted into 2 parts.

  1. Training dataset : This dataset is used to train the model.
  2. Validation dataset : This dataset is used to validate the model.

Step 6

Model creation : In this step model architecture is created. In this step I have created different layers of the model.

Step 7

Traning model : Earlier in step-5 we have splitted the dataset into 2 parts which were Training dataset & Validation dataset. So in this step we will train the model using this Training dataset. Training accuracy of this model is 98.31%.

Step 8

Validating model : Till now we have trained the model now we have to validate it. To validate the model we will use Validation dataset. Validation accuracy of this model is 95.28%.
